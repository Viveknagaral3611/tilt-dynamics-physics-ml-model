# tilt-dynamics-physics-ml-model
Physics-based vehicle tilt simulation and machine learning surrogate model for fast prediction of tilt dynamics during turning maneuvers.


# Vehicle Tilt Dynamics: Physics Model + ML Surrogate

This project develops a physics-based simulator for vehicle tilt during turning and a machine learning surrogate model that approximates the simulator for fast predictions.

---

## Overview

When a vehicle takes a turn, lateral acceleration generates a moment that causes the body to tilt. This project models that behavior using a second-order dynamic system and then trains a neural network to predict the maximum tilt angle directly from input conditions.

The goal is to compare:
- Physics-based simulation (accurate but slower)
- Machine learning surrogate (fast but approximate)

---

## Physics Model

The tilt dynamics are modeled as:

\[
I \ddot{\theta} + c \dot{\theta} + k \theta = m h \frac{v^2}{R}
\]

This represents:
- Inertia resisting motion  
- Damping reducing oscillations  
- Stiffness restoring equilibrium  
- Turning-induced torque  

The system is solved numerically using `scipy.integrate.solve_ivp`.

---

## Dataset Generation

Synthetic data is generated by sweeping:

- Speed \(v\): 5–25 m/s  
- Turn radius \(R\): 20–120 m  
- Damping \(c\): 30–120  
- Stiffness \(k\): 1500–3500  

Total simulations: **1600**

Each sample includes:
- Max tilt  
- Final tilt  
- Max angular velocity  

---

## Machine Learning Surrogate

A neural network (`MLPRegressor`) is trained to predict:

\[
(v, R, c, k) \rightarrow \text{max tilt}
\]

### Model details:
- 2 hidden layers (32 neurons each)
- ReLU activation
- Adam optimizer

## Key Insights

- Physics model → accurate, interpretable  
- ML surrogate → fast, efficient  
- Tradeoff between accuracy and speed  

The surrogate performs best within the training range and may lose accuracy at extreme parameter values.
